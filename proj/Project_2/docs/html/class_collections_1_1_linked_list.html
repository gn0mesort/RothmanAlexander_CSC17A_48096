Detailed Description

template<typename T>
class Collections::LinkedList< T >

Defines a doubly linked list class. Almost certainly not better than std::list. Provides basic functionality like indexing, adding to the front or back of the list, removing nodes, copy construction, and toString. This class forms the basis for the other structures in this namespace
